# bytes-util

> Bytes covert utils.

## Getting started

```bash
npm install bytes-util
```

or

```bash
yarn add bytes-util
```

## API

### `parse`

```js
import bytesUtil from 'bytes-util'

bytesUtil.parse("20MB") // 20480
```

### `stringify`

```js
import bytesUtil from 'bytes-util'

bytesUtil.stringify(20480) // 20 MB
```
